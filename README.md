# Simple Memory Game in Python

The Simple Memory Game is a Python-based project designed to test and enhance memory skills. In this game, a grid of tiles (each representing an image or symbol) is displayed face-down. Players take turns flipping two tiles at a time. If the flipped tiles match, they are removed; otherwise, they are flipped back face-down. ## Installation

2. Install dependencies:
`pip install -r requirements.txt`

## Usage
To run the game and start playing, simply navigate to the pygame folder and execute the following commands:
